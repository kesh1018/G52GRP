------------------------------------------------------------------------------------
HOW TO USE VMS SOFTWARE

Register Smart Card reader
------------------------------------------------------------------------------------

1. Download MYKAD SDK for desktop development here
http://mykadpro.onlineapp.com.my/downloads/MYKADSDK_reg.zip

2. Web development please download the SDK here 
http://mykadpro.onlineapp.com.my/downloads/MYKADSDKWEB_reg.zip

3. Download MYKAD Reader Driver
http://mykadpro.onlineapp.com.my/downloads/FT_SCR2000_Driver.zip

4. Download Registration Key generator.
http://mykadpro.onlineapp.com.my/downloads/myregkey.zip

Note: 
*Make sure you copy/extract all the MYKAD SDK files into your C:\ drive. 
*For example C:\MYKADSDK or C:\MYKADSDKWEB folder


Register DLL to the Operating System
------------------------------------------------------------------------------------
1. Web application (ASP, ASPX, PHP, HTML) 
> double click "registerwebdll.bat" on your Explorer

Load Database into Localhost 
------------------------------------------------------------------------------------
1. Open database folder
2. Go to localhost/phpmyadmin on your browser
3. Create table "dbregistration" & import sql files in the database folder

Load web application
------------------------------------------------------------------------------------
1. Open parent folder
2. Open Xampp/htdocs/ & create new folder
2. Copy files into Xammp/htdocs
3. Then browser to, localhost/(name of new folder)

Sign into VMS
------------------------------------------------------------------------------------
1. Sign in using admin (username : admin@vms.com/password : 12345) or guard (username: guard1@vms.com/password : 12345)

Insert Visitor
------------------------------------------------------------------------------------
1. Load user from MyKad
2. Save

Insert Guard
------------------------------------------------------------------------------------
1. Load user from MyKad
2. Save
