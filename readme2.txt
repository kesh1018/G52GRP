How to use VMS android Application:

Using Emulator
----------------------------------------
1. Download Bluestack Emulator here
http://www.bluestacks.com/download.html

or 

2. Download GenyMotion Emulator here
https://www.genymotion.com

Load php files
----------------------------------------
1. Open Android Application/android-prototype/
2. Copy login-app folder into C:/Wamp/www/
3. Open Wamp

Sign into VMS android Application
-----------------------------------------
1. Open Android Emulator
2. Install apk file
3. Run application
4. Sign in using admin (username : admin@vms.com/password : 12345) or guard (username: guard1@vms.com/password : 12345)
